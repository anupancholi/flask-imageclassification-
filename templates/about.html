    This web application is designed to classify images using a machine learning pipeline built with Python, Flask, and scikit-learn. Users can upload an image, which is then processed through feature extraction and prediction models to identify the most likely categories. The goal of this project is to demonstrate the power of computer vision and machine learning in a simple, interactive interface. The application uses Histogram of Oriented Gradients (HOG) for feature extraction and a trained SGD classifier for prediction, making it a lightweight and efficient tool for real-time image classification tasks.
